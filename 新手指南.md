# 新手建博客完整指南

## 第一步: 准备开发环境

### 1.1 安装Node.js

#### 什么是Node.js?
Node.js是一个运行环境,让你的博客可以在电脑上运行。

#### 如何安装:
1. 打开浏览器,访问: https://nodejs.org/
2. 点击下载LTS版本(长期支持版本,推荐新手使用)
3. 下载完成后,双击安装文件
4. 点击"Next"按照提示完成安装
   - 安装路径可以保持默认
   - 其他选项都勾选默认即可
5. 安装完成后,点击"Finish"

#### 验证安装:
1. 打开"命令提示符"(按Win+R,输入cmd,回车)
2. 输入以下命令并回车:
   ```
   node --version
   ```
3. 如果看到版本号(比如v18.17.0),说明安装成功!

### 1.2 安装MongoDB

#### 什么是MongoDB?
MongoDB是数据库,用来存储你的博客文章、用户信息等数据。

#### 如何安装:
1. 打开浏览器,访问: https://www.mongodb.com/try/download/community
2. 选择Windows版本
3. 点击Download下载
4. 运行安装程序
5. 点击"Next"完成安装
   - 选择"Complete"完整安装
   - 勾选"Install MongoDB as a Service"作为服务安装
   - 数据目录保持默认
6. 安装完成后,MongoDB会自动启动

#### 验证安装:
1. 打开命令提示符
2. 输入以下命令:
   ```
   mongo --version
   ```
3. 如果看到版本号,说明安装成功!

## 第三步: 配置数据库连接

1. 进入后端目录:
   ```
   cd D:\IDE_Project\web_create\my-blog\backend
   ```

2. 复制环境变量文件:
   ```
   copy .env.example .env
   ```

3. 用记事本打开 `.env` 文件,内容应该是这样的:
   ```
   PORT=5000
   MONGODB_URI=mongodb://localhost:27017/myblog
   JWT_SECRET=your-secret-key-change-this-in-production
   NODE_ENV=development
   ```

4. 修改JWT_SECRET(为了安全):
   ```
   JWT_SECRET=my-blog-secret-2024
   ```

5. 保存文件

---

## 第四步: 启动博客系统

### 4.1 启动后端

1. 在第一个命令提示符窗口中(backend目录)
2. 输入以下命令:
   ```
   npm run dev
   ```
3. 看到"服务器运行在端口 5000"提示,说明后端启动成功!
4. **不要关闭这个窗口!**

### 4.2 启动前端

1. 在第二个命令提示符窗口中(frontend目录)
2. 输入以下命令:
   ```
   npm run dev
   ```
3. 看到"Local: http://localhost:3000"提示,说明前端启动成功!
4. **不要关闭这个窗口!**

## 第五步: 创建管理员账户

### 5.1 注册账户

1. 打开浏览器
2. 访问: http://localhost:3000/register
3. 填写注册信息:
   - 用户名: admin (或者你喜欢的名字)
   - 邮箱: your-email@example.com
   - 密码: 123456 (至少6位)
   - 确认密码: 123456
4. 点击"注册"按钮

### 5.2 设置管理员权限

1. 打开命令提示符
2. 输入以下命令进入MongoDB:
   ```
   mongo
   ```

3. 切换到博客数据库:
   ```
   use myblog
   ```

4. 查看用户列表:
   ```
   db.users.find()
   ```

5. 将你的用户设置为管理员(替换成你的用户名):
   ```
   db.users.updateOne({username: "admin"}, {$set: {role: "admin"}})
   ```

6. 退出MongoDB:
   ```
   exit
   ```

## 第七步: 发布第一篇博客文章

### 7.1 创建文章

1. 点击左侧菜单的"文章管理"
2. 点击"新建文章"按钮
3. 填写文章信息:

   **基本信息:**
   - 标题: 我的第一篇博客
   - 分类: 选择"技术分享"
   - 标签: 选择"JavaScript"
   - 状态: 选择"发布"

   **文章内容:**
   - 摘要: 这是我发布的第一篇博客文章,欢迎大家访问!
   - 内容: 可以在这里写你的博客内容,支持HTML格式

   示例内容:
   ```html
   <h2>欢迎来到我的博客</h2>

   <p>这是我发布的第一篇博客文章!</p>

   <p>在这个博客中,我会分享:</p>
   <ul>
       <li>技术教程</li>
       <li>学习心得</li>
       <li>生活感悟</li>
   </ul>

   <p>希望我的文章能对你有所帮助!</p>
   ```

4. 点击"保存"按钮

### 7.2 查看文章

1. 访问: http://localhost:3000
2. 你应该能看到刚才发布的文章
3. 点击文章标题查看详情

## 第八步: 每天如何上传内容

### 8.1 登录后台

1. 访问: http://localhost:3000/login
2. 输入你的邮箱和密码
3. 点击"登录"

### 8.2 发布新文章

1. 点击右上角用户名,选择"管理后台"
2. 点击"新建文章"按钮
3. 填写文章信息:
   - 标题: 你的文章标题
   - 选择合适的分类和标签
   - 写文章摘要(简短介绍)
   - 写文章内容(支持HTML)
   - 选择"发布"状态
4. 点击"保存"

### 8.3 文章内容格式建议

**简单格式:**
```html
<h2>文章标题</h2>

<p>第一段内容...</p>

<p>第二段内容...</p>

<ul>
    <li>要点1</li>
    <li>要点2</li>
</ul>
```

**包含图片:**
```html
<h2>文章标题</h2>

<p>文章内容...</p>

<img src="图片URL" alt="图片描述">

<p>更多内容...</p>
```

### 8.4 管理已发布的文章

1. 在管理后台的"文章管理"中可以看到所有文章
2. 点击"编辑"可以修改文章
3. 点击"删除"可以删除文章

### 8.5 每日发布流程

**建议流程:**
1. 每天固定时间发布(比如早上9点)
2. 先在本地写好文章内容
3. 登录后台,点击"新建文章"
4. 复制粘贴内容,选择分类标签
5. 点击"保存"发布

**快速发布技巧:**
- 可以先创建文章模板
- 使用Markdown编辑器写文章,然后转换为HTML
- 批量准备几篇文章,定时发布

## 常见问题解答

### Q1: 启动后提示"端口被占用"
**解决方法:**
- 修改 `backend/.env` 文件中的PORT(比如改成5001)
- 重启后端服务器

### Q2: 数据库连接失败
**解决方法:**
- 确保MongoDB服务正在运行
- 打开服务管理器,查看MongoDB服务状态
- 如果未运行,右键点击"启动"

### Q3: 前端页面空白
**解决方法:**
- 确保后端和前端都正常运行
- 检查浏览器控制台是否有错误
- 刷新页面

### Q4: 忘记密码怎么办
**解决方法:**
- 直接在数据库中重置密码
- 或者重新注册一个账户
