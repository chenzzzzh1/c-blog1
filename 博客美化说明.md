## 📝 使用说明

### 查看效果

1. **启动前端服务器:**
```bash
cd frontend
npm run dev
```

2. **访问博客:**
   - 打开浏览器访问: http://localhost:3000
   - 刷新页面查看美化效果

### 自定义主题

**修改颜色方案:**
编辑 `frontend/src/assets/styles/global.css` 文件:

```css
:root {
  --primary-color: #409eff;        /* 主色调 */
  --primary-hover: #66b1ff;        /* 悬停色 */
  --text-primary: #303133;        /* 主文本色 */
  --bg-page: #f5f7fa;             /* 页面背景色 */
}
```

**修改导航栏渐变:**
编辑 `frontend/src/components/Header.vue`:

```css
.header {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### 添加自定义图片

1. **准备图片:**
   - 推荐尺寸: 1920x600px (横幅)
   - 文件格式: JPG/PNG
   - 文件大小: < 500KB

2. **上传图片:**
   - 在管理后台发布文章时上传
   - 或直接使用图片URL
